### Enabling GitHub Pages

To make this page accessible online, you need to enable GitHub Pages:

1. Go to your repository on GitHub: `https://github.com/nfjschannel-lgtm/valentine-terminal`
2. Click on **Settings** (top navigation)
3. Click on **Pages** (left sidebar)
4. Under "Source", select **GitHub Actions**
5. The workflow will automatically deploy your page!

After the workflow runs successfully, your page will be available at:
`https://nfjschannel-lgtm.github.io/valentine-terminal/`

### Local Development

To view the page locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/nfjschannel-lgtm/valentine-terminal.git
   cd valentine-terminal
   ```

2. Open `index.html` in your web browser, or use a local server:
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js (requires npx)
   npx serve
   ```

3. Open your browser to `http://localhost:8000`
